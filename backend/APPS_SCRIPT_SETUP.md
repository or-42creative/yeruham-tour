# חיבור לידים ל-Google Sheets + Zapier (Google Apps Script)

הלידים נכתבים לגיליון Google, ואז כל הדאטה מועברת ל-Zapier (משם מנהלים
מיילים/אוטומציות). כל הקוד כבר כתוב ב-[`Code.gs`](Code.gs) — צריך רק להדביק ולפרוס. ~5 דקות.

---

## שלב 1 — צור גיליון
1. היכנס ל-[sheets.google.com](https://sheets.google.com) → **גיליון ריק חדש**.
2. תן לו שם, למשל **"לידים — Yeruham Tour"**. (אין צורך ליצור עמודות — הסקריפט
   יוצר אותן אוטומטית בפעם הראשונה.)

## שלב 2 — הדבק את הקוד
1. באותו גיליון: תפריט **Extensions → Apps Script** (הרחבות → Apps Script).
2. מחק את הקוד לדוגמה שמופיע, והדבק את **כל התוכן** של הקובץ [`Code.gs`](Code.gs).
3. בראש הקובץ ודא ש-`ZAPIER_WEBHOOK_URL` הוא ה-webhook הנכון של Zapier
   (כרגע: `https://hooks.zapier.com/hooks/catch/12914116/4u011ll/`).
4. לחץ **שמור** (אייקון הדיסקט).

## שלב 3 — פרוס כ-Web App
1. כפתור **Deploy → New deployment** (פריסה → פריסה חדשה).
2. ליד "Select type" לחץ על גלגל השיניים ובחר **Web app**.
3. הגדרות:
   - **Execute as:** *Me* (אתה) — כדי שיהיו הרשאות לכתוב לגיליון ולפנות ל-Zapier.
   - **Who has access:** **Anyone** (חשוב! אחרת העמוד לא יוכל לשלוח).
4. **Deploy** → תתבקש **לאשר הרשאות** (Authorize access): בחר את חשבון הגוגל שלך,
   לחץ "Advanced" → "Go to (project) → Allow". זה חד-פעמי.
5. תקבל **Web app URL** שמסתיים ב-`/exec`. **העתק אותו.**

## שלב 4 — חבר לעמוד
1. פתח את [`assets/js/config.js`](../assets/js/config.js) בפרויקט.
2. הדבק את ה-URL בשדה:
   ```js
   LEAD_WEBHOOK_URL: "https://script.google.com/macros/s/AKfy..../exec",
   ```
3. שמור, ו-commit + push ל-GitHub (העמוד יתעדכן תוך דקה).

## שלב 5 — בדיקה
- שלח הרשמת בדיקה מהעמוד. תוך שניות תופיע שורה בגיליון, וה-webhook יופעל ב-Zapier.
- אפשר לבדוק שה-endpoint חי: פתח את ה-`/exec` URL בדפדפן — אמור להחזיר
  `{"ok":true,"msg":"..."}`.

---

## מה קורה מאחורי הקלעים
- כל ליד נכתב כשורה בלשונית **"לידים"**: תאריך, **מועד סיור מבוקש**, שם, טלפון, מייל, יישוב, דעה, ליד רלוונטי, אישור דיוור, מקור, כתובת עמוד.
- מיד אחרי כתיבת השורה, **כל הדאטה מועברת כ-JSON ל-Zapier** (`ZAPIER_WEBHOOK_URL`).
  משם מנהלים את המיילים/האוטומציות — הסקריפט עצמו כבר לא שולח מיילים.
- **עמודת "מועד סיור מבוקש" מתווספת אוטומטית גם לגיליון קיים** (בסוף השורה). שים לב:
  לידים שנשלחו *לפני* ההוספה לא כוללים את המידע הזה (הוא לא נשמר בעבר), אז התא שלהם יישאר ריק.

## עדכון הקוד בעתיד
אם תשנה משהו ב-`Code.gs`: **Deploy → Manage deployments → עיפרון (Edit) →
Version: New version → Deploy**. ה-URL נשאר אותו דבר (לא צריך לעדכן שוב את ה-config).
